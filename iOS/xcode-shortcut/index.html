Trong quá trình phát triển phần mềm, ngoài các kĩ năng về nền tảng, ngôn ngữ đang sử dụng thì việc làm chủ được IDE (integrated development environment) sẽ giúp cho quá trình coding được nhanh chóng và hiệu quả hơn. Sau đây mình xin giới thiệu với các bạn một số shortcuts trong Xcode - IDE để phát triển các ứng dụng iOS.

1. Left - Right - Bottom Menu:

  • Show/Hide Debug Area (Bottom): Shift + Command + Y
  • Show/Hide Navigator (Left): Command + 0
  • Show/Hide Utilities (Right): Option + Command + 0

2. Navigator & Utilities:

Để di chuyển giữa các tabs trong:

  • Utilities (Right Menu): Command Option 1–6
  • Navigator (Left Menu): Command 1–9

3. Quick Open:

Để mở một class, func hay file… nào đó

  • Command Shift o

4. Open Line Number:

Để di chuyển đến line number trong source file đó.

  • Command L

5. Assistance Editor

  • Show: Option Command Enter
  • Hide: Command Enter

6. Indent

  • Control + I

7. Hide Code:

  • Hide: Command Option Left
  • Open: Command Option Right

8. AutoCompletion:

  • ESC or Command + Space

9. Move Line:

  • Left: Command + [
  • Right: Command + ]
  • Up: Command + Option + [
  • Down: Command + Option + ]

Hi vọng bài viết sẽ có ích cho mọi người. Cảm ơn mọi người đã theo dõi.
